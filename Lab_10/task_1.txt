Компьютер\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Microsoft SQL Server\MSSQL15E.LOCALDB\MSSQLServer (смотрим значение параметра LoginMode).

Значение "1": режим аутентификации Windows,
Значение "2": режим аутентификации SQL Server и Windows,
Значение "0": режим аутентификации SQL Server. 
(у нас "2")

Режим аутентификации определяет, как пользователи могут получить доступ к экземпляру SQL Server и какие типы учетных записей могут использоваться для аутентификации. В режиме "Windows Authentication mode" пользователи используют свои учетные записи Windows для аутентификации, а в режиме "SQL Server Authentication mode" они должны использовать учетную запись SQL Server, которая была создана в экземпляре SQL Server. Режим "SQL Server and Windows Authentication mode" позволяет использовать оба типа аутентификации.

Выбор режима аутентификации зависит от потребностей вашего приложения и уровня безопасности, который вы хотите обеспечить. Если ваше приложение использует Windows-учетные записи для аутентификации пользователей, то режим "Windows Authentication mode" может быть более удобным вариантом. Если вы хотите иметь больший контроль над учетными записями и уровнем доступа, то режим "SQL Server Authentication mode" может быть более подходящим выбором.
